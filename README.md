# LabMate AI

![LabMate AI Logo](https://via.placeholder.com/1200x320?text=LabMate+AI)

LabMate AI is a modern laboratory workspace designed for scientists, researchers, and QA teams. It combines calculators, SOP generation, a lab notebook, and dashboard insights in a single streamlined experience.

## ✨ Features

- Interactive calculator hub for chemistry, molecular biology, cell biology, microbiology, QC, and unit conversion
- Fully functional dilution calculator with validation, copy, save, and clear actions
- Lab notebook for recording experiments, results, and notes
- LocalStorage-powered dashboard with recent calculations, recent notes, and favorites
- Responsive interface designed for desktop and mobile use
- Clean, laboratory-inspired user experience

## 🚀 Installation

1. Clone the repository
   ```bash
   git clone https://github.com/thrudfla/labmate-ai.git
   cd labmate-ai
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

4. Open the app in your browser at the local Vite URL.

## 🛠️ Tech Stack

- React
- Vite
- JavaScript / JSX
- LocalStorage for persistence
- CSS-in-JS style objects
